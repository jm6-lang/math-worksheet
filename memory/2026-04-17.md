## AI Website Cloner Template 学习与部署

### 背景
用户要求学习 GitHub 项目 AI Website Cloner Template (JCodesMore/ai-website-cloner-template)，
将其技能内化并部署到 OpenClaw。

### 项目信息
- **Repo**: github.com/JCodesMore/ai-website-cloner-template
- **Star**: 11,711（截至 2026-04-17）
- **技术栈**: Next.js 16 + React 19 + TypeScript + Tailwind CSS v4 + shadcn/ui
- **MIT License**

### 核心原理
通过 `/clone-website <URL>` 指令，AI Agent 执行 5 阶段流水线：
1. **Reconnaissance** — 截图、设计 Token 提取、交互 Sweep（scroll/click/hover/responsive）
2. **Foundation Build** — 字体 → 全局 CSS（颜色/间距）→ TypeScript 类型 → SVG 提取 → 资源下载
3. **Component Spec + Dispatch** — 每个 section 写 `.spec.md` → 分发给 builder agents
4. **Merge + QA** — 合并、验证 build、逐像素视觉对比
5. **Visual QA** — 强制对比验证，禁止跳过

### 关键技能点（已学透）
- `getComputedStyle()` 精确提取 CSS 值，禁止估算
- 交互模型识别（scroll-driven / click-driven / hover-driven / time-driven）优先于写代码
- 所有状态都要提取，不只是默认状态
- Spec 文件 = 信任契约，不写 spec 不分发 builder
- 复杂度拆分规则：spec >~150 行必须拆分
- Layered images：背景色 + 前景图 + overlay = 3 个资源都要抓
